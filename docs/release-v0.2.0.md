# v0.2.0 — Modern Dashboard for RelayForge

## What's New

- **Modernized Dashboard UI** — clean card-based layout with left sidebar navigation
- **New Overview onboarding** — hero section, quick setup cards, health metrics, next steps wizard
- **Dedicated Combo Models page** — core feature highlighted with step-by-step candidate visualization
- **Clients configuration page** — one-click copy configs for CC Switch, opencode, Codex, Cline, generic OpenAI-compatible
- **Improved Usage page** — stat cards, provider/model ranking, error distribution, recent requests table
- **Card-based Provider view** — each provider shown as a card with key status, health, and capabilities
- **Reorganized navigation** — Overview, Providers, Combo Models, Clients, Routes, Usage, Tools, Settings, IDE
- **Empty states** — friendly guidance when no providers, requests, combos, or errors exist
- **Mobile responsive** — sidebar collapses on small screens
- **Dashboard HTML smoke tests** — verifying brand, layout, and content
